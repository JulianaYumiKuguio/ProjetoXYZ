# ProjetoXYZ
ProjetoXYZ - Teste VB6-SQL

Entregáveis

1. Código-fonte da aplicação (via GitHub).
   Segue código-fonte de todos os forms, modulos criados no VB6.
   
2. Script SQL completo (tabelas, procedures, functions, views e dados de exemplo).
   Segue 7 scripts de sql com ordenação númerica para rodar na base de dados.
   Para fins de testes, criei uma rotina que cria autormaticamente 5.000 mil registros com dados aleatórios na tabela.

3. Exemplo de relatório gerado.
   reltransações.rpt (crystal reports)
   reltransações.pdf (layout)
   
4. Instruções
   No arquivo "Config.ini", deve-se parametrizar as informações da conexão com o SQL SERVER (server, user, password, databse)
   Neste mesmo arquivo, deve-se parametrizar também o caminho do diretório do RPT ("reltransações.rpt") e arquivo de log de erros ("MinhaApp_LogErros.txt")
   E por fim gerei o executável do projeto ("ProjetoXYZ.exe")
